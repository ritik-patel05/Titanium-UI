Avatar

A small highly usable image component

Avatars focus on a subject, using a simple background. They can represent a user or a brand (with a logo or branded graphic).

They come in 4 sizes:

  • Small
  • Normal
  • Medium
  • Large

Image avatars can be created by using standard img HTML element inside parent avatar container.
