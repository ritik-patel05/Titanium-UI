Documentation

Everything you need to create a website with Titanium-UI.

Alert

An alert displays a short, important message in a way that attracts the user's attention without interrupting the user's task.

Avatar

Avatars are found throughout design with uses in everything from tables to dialog menus.

Badge

Badge generates a small badge to the top-right of its child(ren).

Button

Buttons allow users to take actions, and make choices, with a single tap.

Card

Cards contain content and actions about a single subject.

Image

Responsive and useful classes for your image.

Input-Form

Text fields let users enter and edit text.

Select-Form

Select components are used for collecting user provided information from a list of options.

Typography

Use typography to present your design and content as clearly and efficiently as possible.

Snackbar

Snackbars provide brief notifications. The component is also known as a toast.

Navbar

The top Navbar provides content and actions related to the current screen. It's used for branding, screen titles, navigation, and actions.
